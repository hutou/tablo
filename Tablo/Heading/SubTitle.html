    Overview

    struct SubTitle creates a subtitle for the table (but displayed only if a title has also been defined)

    def self.new(value : CellType = nil, frame : Frame | Nil = nil, alignment : Justify = Config.heading_alignment, formatter : Cell::Text::Formatter = Config.heading_formatter, styler : Cell::Text::Styler = Config.heading_styler) #

    Returns an instance of SubTitle

    Example:

    Tablo::Table.new((1, 2, 3],
  subtitle: Tablo::Heading::SubTitle.new("My subtitle", frame: Tablo::Frame.new(1, 1),
    alignment: Tablo::Justify::Left)

    All (named) parameters are optional, with default values

    • value: type is CellType
      Default value is nil
      This is the subtitle's display content.
    • frame: type is Frame?
      Default value is nil
      If a Frame instance is created, the subtitle is framed.
    • alignment: type is Justify
      By default, as defined in Config.heading_alignment (but this can be modified), subtitle content is centered.
    • formatter: a Proc whose type is Cell::Text::Formatter
      Default value is set by Config.heading_formatter
    • styler: a Proc whose type is Cell::Text::Styler
      Default value is set by Config.heading_styler
